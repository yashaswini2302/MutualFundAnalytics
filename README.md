# 📊 Mutual Fund Analytics Platform

## Overview

The Mutual Fund Analytics Platform is an end-to-end data analytics project developed during the **Bluestock Fintech Internship**. The project automates the extraction, transformation, analysis, and visualization of mutual fund data using Python, SQLite, SQL, and Power BI.

It provides investors with meaningful insights into fund performance, risk assessment, portfolio trends, and investment recommendations through interactive dashboards and advanced financial analytics.

---

## Objectives

- Develop an automated ETL pipeline for mutual fund datasets.
- Store cleaned data in SQLite.
- Perform Exploratory Data Analysis (EDA).
- Analyze mutual fund performance using financial metrics.
- Build interactive Power BI dashboards.
- Implement advanced analytics such as VaR, CVaR, Rolling Sharpe Ratio, HHI, Cohort Analysis, and Fund Recommendation.
